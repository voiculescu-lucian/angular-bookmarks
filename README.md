# AngularBookmarks

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.0.4.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.

## Task Details:

1. This is a Single Page Application called Bookmarker.
2. List Bookmarks UI - filtered by today, yesterday, and older with edit facility.
3. Edit Bookmark UI - the edit link above makes you edit the bookmark you want. After submission, you should be redirected to the list page (above).
4. Create Bookmark UI - lets you add a new bookmark. After submission, you should be redirected to the list page (above).
5. Filter List bookmarks UI - lets you filter the list of bookmarks using a Fuzzy search on top.

## Building

To build the project run:

```bash
ng build
```

This will compile your project and store the build artifacts in the `dist/` directory. By default, the production build optimizes your application for performance and speed.
